### c) automoottorix

- siirrytään ansiblen kansioon `cd ansible` komennolla ja muokataan `site.yml` lisäämällä sinne nginx:
<img width="307" height="151" alt="kuva" src="https://github.com/user-attachments/assets/9cf34240-4b03-4f0d-85c8-a0c8075b4265" />

- sen jälkeen tehdään uusi kansio, ensin siirtymällä `roles`, ja lisäämällä sinne nginx komennolla `mkdir nginx`
- lisätään sinne kolme eri kansiota: `mkdir files`, `mkdir handlers` ja mkdir `tasks`
- tehdään handlersiin tiedosto `main.yml` ja lisätään sinne ohjeen mukaan
```
- name: restart nginx
  systemd:
    name: nginx
    state: restarted
```
- siirrytään sen jälkeen `tasks` kansioon ja luodaan sinne `main.yml`
- kirjataan sinne seuraava:
```
- name: install nginx
  apt:
  name: nginx
  state: present
  update_cache: yes

- name: start nginx
  service:
  name: nginx
  state: started
  enabled: true
```
- files kansioon ei tarvinnutkaan tässä tehtävässä koskea, jos käsitin tehtävän oikein.

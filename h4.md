### a) Räpylä

- Asennusvaiheen ehdin tehdä jo tunnilla. Valitsin demoniksi PostGresin, sillä siihen löytyi ohjeet helposti.
- Asensin demon komennolla `sudo apt-get install postgres`
- asennuksen jälkeen käynnistin demonin, ja loin sinne databasen ja käyttäjän:
```
sudo systemctl start postgres
sudo -u postgres createdb meas
sudo -u postgres createuser meas
```
